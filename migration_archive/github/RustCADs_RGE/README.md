# RustCADs/RGE GitHub Archive

Static GitHub API export captured before continuing development under
`drhalilmurattasci/RGE` and deleting the old `RustCADs/RGE` repository.

The archive is intentionally raw JSON so the original GitHub issue and pull
request metadata remains queryable after the old repository is removed.

See `manifest.json` for the capture timestamp, counts, and file map.
