## Planner Notes

This issue intentionally supersedes the earlier single-manifest load-only shape by first fixing the six existing golden manifests so they are valid `Project` values. The test should still stay focused on `simple-scene`; the other five manifests are acceptance-covered by the canonical gate and by the fact that they use the same schema reconciliation pattern.

Prefer a small integration test that uses existing public `rge_data` types and `ron::from_str`.
