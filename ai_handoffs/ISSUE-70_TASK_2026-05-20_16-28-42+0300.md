## Goal

Add a focused Phase 8 unit-test smoke proving `ScriptGraph` composes with the
shared graph-foundation structural diff path. The test should snapshot an
initial script graph, add exactly one script node and one script edge, diff the
old and new snapshots with `GraphDiff::between`, and prove that only those two
additions are reported.

## Deliverables

- Add one unit test named
  `structural_diff_reports_added_script_node_and_edge` in
  `crates/script-graph/src/lib.rs`.
- Keep the test inside the existing `tests` module and reuse the private
  `g.graph` substrate access pattern already used by the snapshot round-trip
  test.
- Extend test-only imports only as needed, for example by importing
  `GraphDiff` alongside the existing graph-foundation test imports.
- The test must:
  - Build an initial `ScriptGraph` with at least two script nodes and one
    script edge.
  - Snapshot that initial graph using `GraphSnapshot::from_graph(&g.graph)`.
  - Mutate the same graph by adding exactly one new script node and exactly one
    new script edge.
  - Snapshot the mutated graph and call `GraphDiff::between(&old_snapshot,
    &new_snapshot)`.
  - Assert `diff.added_nodes` contains exactly the new `ScriptNode` payload by
    `NodeId`.
  - Assert `diff.added_edges` contains exactly the new `EdgeRecord<ScriptEdge>`
    by `EdgeId`, including source, destination, and edge key payload.
  - Assert `removed_nodes`, `removed_edges`, `changed_nodes`, and
    `changed_edges` are all empty.
- Write and finalize the normal `EXECUTION_REPORT` packet after execution.
