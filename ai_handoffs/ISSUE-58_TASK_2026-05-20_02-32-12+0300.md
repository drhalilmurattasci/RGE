- Expose the new widget module from `crates/editor-ui/src/widgets/mod.rs` with a minimal public module declaration.

- The surface should include a lightweight widget shell, for example `NodeGraphWidget`, and a borrowed model, for example `NodeGraphModel`, whose node and edge records copy ids/endpoints while borrowing display labels from the adapter views.
