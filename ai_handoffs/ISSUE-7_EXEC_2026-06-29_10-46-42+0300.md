## What I Changed

### Source

- `crates/editor-ui/src/menus/keybinding.rs`: added public data-only
  `KeybindingTarget`, `KeybindingOverride`, `KeybindingOverrides`, and
  `KeybindingDiagnostic` types. The collection is in-memory, defaults empty,
  preserves insertion order, and never reads/writes settings, host state, disk,
  plugin runtime, or Cargo resources.
- `crates/editor-ui/src/menus/mod.rs`: added `pub mod keybinding` and re-exported
  the new keybinding API types without renaming existing exports.
- `crates/editor-ui/src/menus/registry.rs`: made `resolve` delegate to
  `resolve_with_keybinding_overrides` with empty overrides; added
  `resolve_with_keybinding_overrides` to apply remap/unbind operations to cloned
  resolved entries before accelerator/command/conflict indexes are built; added
  non-fatal `keybinding_diagnostics` for targets unknown before visibility
  filtering.

### Tests

- `empty_keybinding_overrides_preserve_default_resolve_behavior`: pins default
  resolve equivalence for empty overrides, 19 accelerators, zero conflicts, menu
  signatures, and shortcut command lookups.
- `keybinding_remap_applies_to_one_resolve_without_mutating_registry`: remaps
  File/Open for one resolve and verifies a later normal resolve still uses the
  original default shortcut.
- `keybinding_unbind_removes_executable_shortcut_but_keeps_entry_visible`:
  unbinds Delete Current CAD Cuboid while preserving the visible entry, command,
  label, enablement, and passive hint state.
- `keybinding_remap_conflict_is_nonfatal_and_suppresses_execution`: remaps
  File/Open onto Ctrl+S, verifies deterministic conflict data, display lookup,
  and `enabled_command_for_shortcut` suppression.
- `unknown_keybinding_target_reports_diagnostic_without_changing_resolve`:
  verifies unknown target diagnostics do not change the menu tree, accelerator
  count, or conflict set.
- `known_hidden_keybinding_target_does_not_emit_unknown_diagnostic`: pins the
  acceptance rule that registered targets are known even when visibility filters
  hide them.
