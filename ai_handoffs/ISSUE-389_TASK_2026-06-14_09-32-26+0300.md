## Deliverables

- A lifecycle-local focus-loss path in `EditorShell::window_event` for
  `WindowEvent::Focused(false)` that cancels active viewport orbit and pan
  drag state.
- Cursor-grab release logic that requests `CursorGrabMode::None` through the
  existing viewport drag cursor-grab helper only when at least one viewport
  drag was active before focus loss.
- Focused unit coverage in `crates/editor-shell/src/lifecycle/tests.rs` for
  active orbit focus-loss cancellation, active pan focus-loss cancellation,
  combined active drag cancellation/release behavior if needed, and the
  no-active-drag no-release case.
- Status/task docs marking task 135 complete after implementation, without
  appending task 136.

## Acceptance Criteria

- `WindowEvent::Focused(false)` cancels any active viewport orbit drag and any
  active viewport pan drag in the existing `window_event` match.
- If at least one viewport drag was active before focus loss, focus loss emits
  exactly the required viewport cursor-grab release request after both drag
  states are idle. If no viewport drag was active, focus loss emits no new
  cursor-grab release request.
- `WindowEvent::Focused(true)` preserves existing behavior and does not start,
  stop, or release viewport drags.
- The implementation reuses the existing drag-stop/cursor-release helpers or a
  narrow private helper in `mod.rs`; it does not change drag math,
  `CursorGrabMode::Confined` start behavior, failed-grab behavior,
  dual-drag release ordering, wheel zoom, frame-all, or face-pick gates.
- Focused tests cover active orbit/pan focus-loss cancellation plus the
  no-active-drag no-release case. Existing `viewport_drag_cursor_grab`,
  `viewport_right_button_orbit`, and `viewport_middle_button_pan` tests continue
  to pass.
- `.ai/dispatch.tasks.md`, `Status.md`, `HANDOFF.md`, `plans/BASELINE.md`, and
  `change.md` record task 135 as done; task 136 is not appended.
- `git diff --name-only` shows only MAY-edit files plus generated ISSUE-389
  artifacts, and `git diff --check` is clean.
