## Deliverables

- `KeybindingDiagnostic` has two new non-fatal data variants:
  `NoOpRemap { target: KeybindingTarget, shortcut: Shortcut }` and
  `RedundantUnbind { target: KeybindingTarget }`.
- `resolve_with_keybinding_overrides` emits those variants only for overrides
  that match a resolved visible entry and leave its executable `shortcut`
  unchanged.
- Resolve behavior is otherwise unchanged: real remaps and unbinds still apply,
  unknown targets remain `UnknownTarget`, known hidden targets stay silent,
  conflicts remain non-fatal conflict data, and executable conflicted shortcuts
  remain suppressed.
- `crates/editor-ui/tests/menus_ordering.rs` covers no-op remap, redundant
  unbind, real remap with no new diagnostic, real unbind with no new diagnostic,
  unchanged `UnknownTarget`, and the default menu's 19 accelerators / zero
  conflicts invariant.
- `.ai/dispatch.tasks.md` appends exactly one task 175 source/docs-read-only
  audit block and carries forward the required self-re-arm text verbatim.

## Acceptance Criteria

- `NoOpRemap` is emitted when a remap override targets a resolved visible entry
  whose current executable shortcut already equals the requested shortcut.
- `RedundantUnbind` is emitted when an unbind override targets a resolved
  visible entry whose current executable shortcut is already `None`.
- The new diagnostics are not emitted for real value-changing remaps, real
  value-changing unbinds, unknown targets, or known-but-hidden targets.
- Existing `UnknownTarget` behavior is unchanged and remains non-fatal.
- The shipped default menu still resolves to exactly 19 executable accelerators
  and zero conflicts.
- No edit lands outside the allowed files, except current dispatch artifacts
  produced by the orchestrator/finalizer.
- Task 175 is appended exactly once and is audit-only: docs/source reads are
  allowed, its MAY edit list is limited to `.ai/dispatch.tasks.md`, `Status.md`,
  `HANDOFF.md`, `plans/BASELINE.md`, and `change.md`, and it forbids Rust
  source, tests, and automation edits.
