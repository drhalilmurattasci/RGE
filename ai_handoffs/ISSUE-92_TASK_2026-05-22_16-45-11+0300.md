## Goal

Run the GitHub issue #92 read-only preflight for the W16 `rge-asset-store` integration shape. Audit the current cache seams in `crates/io-gltf/` and `crates/io-image/`, compare them against `rge-asset-store`'s `Cache` trait and `LocalCache`, and determine whether `kernel/asset-view` becomes a real consumer or remains a future descriptor substrate. This dispatch produces only an evidence-backed recommendation for the next smallest implementation dispatch; it does not implement the integration.
