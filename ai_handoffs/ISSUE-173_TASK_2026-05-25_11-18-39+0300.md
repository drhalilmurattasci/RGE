likely implementation is to extend the existing simple-scene integration test
helper so it reads `.rge-project` as `rge_data::Project`, resolves the manifest
scene reference relative to that manifest, parses the scene, imports it through
`load_scene_into_world`, and then adds the tick assertion on that imported
world.

names (`Camera`, `KeyLight`) or by the canonical IDs already pinned in the
golden scene, as long as the assertions are made against the entities imported
into the `World` by the loader.
